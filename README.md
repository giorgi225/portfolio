## Getting Started
Clonse project & navigate to root folder
```bash
git clone https://github.com/giorgi225/portfolio.git
cd portfolio
```

run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
